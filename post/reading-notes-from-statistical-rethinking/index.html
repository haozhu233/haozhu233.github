Reading notes from statistical rethinking

2018-08-10
readings

I always have an anxiety to learn more about bayesian analysis but I have never put in any real action. Earlier today, I saw Davis Vaughan’s tweet on twitter asking basically the same question that has been bugging me for a long time. I saw many people are talking about the “statistical rethinking” book so I think, “umm, maybe I should give it a try”.

https://twitter.com/dvaughan32/status/1027699962832986115

Luckily, I found this book in the little library we maintained at my workplace. I read the preface + chapter 1 and I found myself really enjoying reading this book. I will try to put down some notes here. These notes may include some conclusions or just some good thought I found in the book.

Preface

The book uses much more computer code than formal mathematics. Even excellent mathematicians can have trouble understanding an approach, until they see a working algorithm. This is because implementation in code form removes all ambiguities.

Same as “deep learning with R”. Good to know I’m not the only one who hates to read math. :P
